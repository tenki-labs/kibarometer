# kibarometer

Public-facing labour-market dashboard for Norway. Fetches data from NAV
(the Norwegian Labour and Welfare Administration), runs our own analysis,
and publishes cite-able snapshots for journalists at
[kibarometer.no](https://kibarometer.no).

See [`scaffolding.md`](./scaffolding.md) for the build blueprint and
[`CLAUDE.md`](./CLAUDE.md) for the playbook.
